非典型大選分析

2020 大選之各鄉鎮政黨支持狀況地圖視覺化

你支持的政黨，選得怎麼樣？

一般選舉後的分析報導大多著重在贏家，不過其實我們還有很多角度可以切入來瞭解。這次就讓我們用鄉鎮地圖的形式來看看選舉數據吧！我們在這個作品裡呈現了以下的數據：

  • 投票率 / 廢票率
  • 政黨票跑票狀況
  • 各政黨的鄉鎮級支持度分布狀況

投票率與廢票率

全國投票率高的區域以台北、桃園、新竹、台中、高雄等大都市為主，不過在屏東霧台、高雄茂林亦有較高的投票率。廢票率則以彰化、雲嘉一帶較高。金門雖然有相當低的投票率，但廢票率亦相對低。

投票率 100% * 投票數 / 具投票權人口數
為表現細節，色階非線性對應，以全國平均分界
廢票狀況 100% * ( 投票數 - 有效票 ) / 投票數
越紫越廢

政黨票與總統票差距

直覺上來說，若選民支持特定候選人，他應該也會支持候選人所屬的政黨，但現實中並非如此。由於政黨選擇較總統候選人來得多很多，一般來說同一選區的總統票都會較政黨票來得少。我們使用以下公式來表現政黨票與總統票票數與對應的政黨票票數間的差距：

公式: 100% × (陣營政黨票 - 總統票) / 總統票

這個數字在不同範圍有不同的含意：

  • 當數字小於 100% 時, 代表政黨票比總統票少。也就是說雖然我支持你，但我不支持你的政黨。數字越小，這個狀況越嚴重。
  • 當數字接近 100% 時，代表對總統與政黨的認同接近一致。
  • 當數字大於 100% 時，代表政黨票比總統票多。也就是說雖然我支持你的政黨，但對於你我真的投不下去。數字越大，這個狀況越嚴重。

換句話說，

<0%

挺人不挺黨

這個政黨讓人投不下去

>0%

挺黨不挺人

這個候選人讓人投不下去

綠營差距

政黨票大都少總統票 20% 以上，在東部、金門甚至達到 -50% 以上。顯然在政黨上需要加油。"要不是蔡英文，我才不要投給你！"

藍營差距

普遍於 -10% 左右，政黨與總統票較民進黨一致，但嘉義義竹出現政黨票大於總統票的現象。"就算我是鐵藍粉，韓國瑜我還是投不下去.."

為突顯唯一正值，>0% 色調有強化而非線性。

各鄉鎮特定政黨支持度

選擇特定政黨, 以鄉鎮區等級呈現該政黨的支持率熱度地圖。

想知道哪個地區的統促粉比例最高嗎？

此地圖以熱度圖方式呈現各政黨在每個鄉鎮的支持度相對於其全國平均值的高低，越亮代表該鄉鎮支持度超出全國平均越多：。

鄉鎮 A 亮度 = (鄉鎮 A 的政黨 P 支持率) / (全國的政黨 P 支持率)
試著切換政黨試試吧

一般新聞報導關注的還是以熱門政黨為主，因此我們不容易看到小黨的支持度狀況。這個地圖納入 19 個政黨票的政黨，並以熱度圖呈現其分布。其中一些有趣的分布，你可以切換到以下政黨來觀察：

  • 台灣基進 - 集中在高雄沿海區域
  • 安定力量 - 在南投縣信義鄉獲得 7% 支持度 ( 全國平均的 11 倍 )
  • 合一行動聯盟 - 在蘭嶼獲得 6.5% 支持度 ( 全國平均的 52.7 倍 )
  • 台澎黨 - 不意外的在澎湖有高支持度
  • 一邊一國 - 熱度集中在雲林以南 / 台南以北
  • 時力、台民黨與民進黨有高度重疊
  • 親民黨 - 分布相當平均


相對支持度

前一小節是各個政黨與自己的全國支持率來比較，但是同一個鄉鎮內，比例與全國平均差異最大的又是哪個政黨呢？我們將同一鄉鎮內各政黨支持度與全國平均值相減，並取出增加最多 / 減少最多的兩個政黨，分成兩張圖呈現。

簡單來說，「在這個區域內，相對於全國平均值，哪個政黨有更多的支持 / 更少的支持度？」
